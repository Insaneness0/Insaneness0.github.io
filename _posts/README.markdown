# 네트워크 유량
## 용어 정리
* Source: 유량의 시작점 
* Sink: 유량의 도착점
* Capacity: 용량
* Flow: 유량
* c(a,b):  a에서 b로 보낼 수 있는 용량
* f(a,b):  a에서 b로 흐르는 유량
## 개념 설명
1. 용량의 제한: 유량은 용량보다 항상 작거나 같다.
2. 유량의 대칭: 예를 들어 a에서 b로 유량의 흐르는 것과 b에서 a로 유량이 흐르는 것은 같은 것이다.
3. 유량의 보존: 들어오는 유량과 나가는 유량의 크기는 같다.
